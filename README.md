# Behavioral AI Analyzer
Analyze how text is written—not just what it says.

Behavioral AI Analyzer is a browser-based application that evaluates the writing process instead of relying only on the final text. It captures behavioral signals such as typing speed, editing activity, pauses, typing rhythm, and paste events to generate explainable insights.

Unlike traditional AI detectors, this project focuses on how content is created rather than only analyzing the finished text.

Features
Real-time typing behavior analysis
Paste event detection
Typing speed (WPM)
Editing activity tracking
Pause detection
Interactive dashboard
Human vs AI heuristic scoring
Privacy-first (100% local)
Technologies
HTML5
CSS3
JavaScript
